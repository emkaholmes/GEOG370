GIS Assignments from GEOG 370: Introduction to GIS

Emma Holmes

These GIS Assignments were all completed using QGIS.

Homework 1

I created a map of Mecklenburg County to practice using vector data.

Homework 2: My Life in Chapel Hill

I created a map showing locations and routes on UNC's campus that I frequent.

Homework 3: World Projections

I mapped different projections on a global scale to see what aspects of the map were distorted and which were kept.

Homework 4: Georeferencing Chapel Hill

I practiced georeferencing an image of Chapel Hill using control points, I then uploaded it to GitHub as a tiled web map.

Homework 5: Georeferencing Seattle

I practiced georeferencing using control points but this time with an old satellite image of Seattle. I also uploaded it as a tiled web map.

Homework 6: Census Data

I created a chloropleth map showing childbirth in North Carolina.

Homework 6.2: Ratios and Classifications

I fixed my previous chloropleth map by creating a ratio to better look at fertility in North Carolina. I then used this map to list pros and cons for three different classification methods: Equal Intervals, Natural Intervals, and Quantile.

Homework 7: Chloropleth Map

I created a chloropleth map showing the percentage of binge drinking per county in Utah. If I redid this map I would change the color scheme so black was the highest percentage rather than the lowest. I would also make the lines between counties a grey or white color to make it easier to distinguish different counties.

Homework 8: Geoprocessing Analysis

I created a map that answered the question "How many health sites are in each city with a population over 100,000 in Bosnia and Herzegovina?"

Homework 9: Google Earth Engine

This is the only assignment that was not completed on QGIS and was instead completed on Google Earth Engine. I looked at health sites in the Sarajevo canton of Bosnia and Herzegovina and used a buffer to see if there were any areas of the canton that lacked easy access to health sites.
